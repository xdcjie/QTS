# Project Manifest

This repository scaffold contains:

- Root project rules: `AGENTS.md`
- Codex config and prompts: `.codex/`
- Design documents: `docs/`
- Implementation plan and acceptance criteria: `docs/plan/`
- Python backend skeleton: `backend/src/qts/`
- Module-local rules: `backend/src/qts/**/AGENTS.md`
- Test layout: `tests/unit`, `tests/integration`, `tests/anchor`
- Example user strategies and configs: `examples/`
- Runtime configs: `configs/`
- Scripts: `scripts/`
- Docker placeholders: `docker/`

Do not keep temporary planning files such as `quant_trading_system_md_file_list.md` in the project. Use `MANIFEST.md` and `docs/README.md` instead.
