# Archive

This directory is for historical documents that are intentionally retained but should not be interpreted as current project direction.

Archived material should explain why it remains useful, for example:

- historical context for a design decision;
- evidence for a completed migration;
- a superseded plan that is referenced by an issue or PR;
- a postmortem or retrospective.

Do not put generated runtime output here unless it has been promoted into durable evidence with context.
